How to use

Page contain three converters with buttons. Each buttons react with converter above.

  • Button CONVERT calculate value typed in input
  • Button CHANGE SIDES swaps converted values which are displayed above input field
  • Button RESET wipe data and bring values to deafult

Conversion results are displayed just under input fields.
